Ukhupholisa
healthcare application

This application, was created with visual studio, it makes use of C#, a ms sql server database and .Net framework that makes up the gui

This application is a healthcare application that allows the company Ukhupholisa Healthcare, to keep track of their clients and their various policies. The application can add new clients with specific policies. The application can create unique ids codes for each policy and based on said codes it makes it easier to find out the policy the clients is on.

The application has 5 main pages. The first one is used by the employees working at their desks to receive call from clients that are there to claim. Th esecond part id the medical department which has a database that has a list of all medical conditions and treatments coverd by the company. In the thirs page we have the providers page which helps the employees involved with creating relationships with various treatment and medical facilities, to maintain those relationships and monitor contracts. The fourth page is the policies page, where new policies are created by the company, this would also be the screen where policies would be maintained and deleted. The final page is meant to analyse the profit that has been generated by the clients and weigh that against the expenses of providing healthcare. In the final page teh copmany is able to edcide bsed on the data what policies to keep and which ones to discard.
