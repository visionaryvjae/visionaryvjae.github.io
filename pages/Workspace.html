Job finding application

Created using React Native and Firebase

This application is a job finding application that I was apart of. This application was a job finding application that allows people wihtin a similar area to create and perform jobs.

The application functions through saving different jobs and profiles on a database which is then displayed on the home screen similar to a social media application that shows all current jobs available. The application has a client side and a employer side. the client side is the people that are looking to complete jobs and the employer side is people creating jobs.
