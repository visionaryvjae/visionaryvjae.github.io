Music Knowledge
tester game

My Music application, created using visual studio and C#

This application is a game tha can be played with a maximum of 2 players. The application shows questions that ask the users about song names or artists to test their knowledge of music. The questions are classified by genre and after reaching a certain number of points the game ends.

The application is an older project and makes use of threading as a means to keep track of the different users that are currently playing. The application was made completely with C# and does not make use of a database but rathere textfiles to store the data on the users and song information.
